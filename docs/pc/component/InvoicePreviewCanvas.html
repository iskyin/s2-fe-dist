# ··· InvoicePreview（发票预览）

# 属性

参数 说明 类型 可选值 默认值 必传
type 发票类型 Number 0 增值税发票 1 区块链电子发票 0 N
buyer 购买方 Array - - N
list 商品清单 Array - - N
total 商品清单总计 Array - - N
bak 备注 String - - N
seller 销售方 Array - - N
bottom 底部 签名区域 Array - - N
