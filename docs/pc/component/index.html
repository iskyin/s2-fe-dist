# 开发指南

此组件库是财经线基于 vue + TDesign 的 PC 库组件库，此库是由项目中分离而来，对组件做的二次封装，目的是为了产品的 SaaS 化，提供拿来即用，面向小众的二次改造后的组件库

注意: 项目原组件库引用的`Ant-Design-Vue`,为了拥抱我司的开源协同，股将项目中的组件库改为了 TDesign。因 TDesign 小伙伴的产品还处于持续迭代中，一来为了避免重复造轮子，二来避免 TDsign 迭代影响我们现有代码，保证我们的项目稳定，故有些组件是直接将 TDsign 的源码引入进来的。待 TDesign 稳定后统一给替换掉。

# 优点

  1. 统一设计标准
  2. 面向 SaaS 化，引入即用
  3. 组件优化，交给专业第三方维护。

# 特别提示

  1. 此组件库基于 Vue，请务必学习 Vue 相关的 Component API (opens new window) 和 组合式 API (opens new window)
  2. 推荐您直接使用脚手架s2-vue3 (opens new window)整体安装应用项目
  3. 项目基于 Vue + Typescript + jsx + Webpack 请提前学习相关知识
