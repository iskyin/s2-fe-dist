# ··· JSX 编码规范

# 自闭合

// bad
<Header class="head"></Header>

// good
<Header class="head" />

# 属性对齐

// bad
<Header class="head" onchange="change()" ></Header>

// good
<Header class="head"
        onchange="change()" >
/>

# 命名规范

  1. 文件夹名称，多个单词时，使用小驼峰命名法，如：notFound
  2. 组件库命名，使用大驼峰命名法，如：InvoicePreviewCanvas
  3. 页面需要创建文件夹，文件夹下是 index 命名的 tsx 文件和 leass 文件
  4. api 接口方法名，在 api 文件夹下创建文件，url 统一存放到 url.ts
  5. props 属性名，前缀：props，如：propsAppId
  6. computed 计算属性名，前缀：computed，如：computedAppId
  7. emit 事件名，前缀：emit，如：emitInput
