# 开发指南

# 安装

# npm 安装

推荐使用 npm 的方式安装，它能更好地和 webpack 打包工具配合使用。

tnpm install  @tencent/s2pc-ui -S
# 或
yarn add @tencent/s2pc-ui -S

# 快速上手

本节将介绍如何在项目中使用 s2pc-ui。

# 引入 S2PC-UI

你可以引入整个 Element，或是根据需要仅引入部分组件。我们先介绍如何引入完整的 Element。

# 完整引入

在 main.js 中写入以下内容：

import Vue from 'vue';
import S2PC from '@tencent/s2pc-ui';
import '@tencent/s2pc-ui/lib/index.css'; // 样式文件需要单独引入
import App from './App.vue';

Vue.use(S2PC);

new Vue({
  el: '#app',
  render: h => h(App)
});

# 按需引入

借助 babel-plugin-component，我们可以只引入需要的组件，以达到减小项目体积的目的。

首先，安装 babel-plugin-component：

tnpm install babel-plugin-component -D
# 或
yarn add babel-plugin-component -D

然后，将 .babelrc 修改为：

{
  "presets": [["es2015", { "modules": false }]],
  "plugins": [
    [
      "component",
      {
        "libraryName": "@tencent/s2pc-ui",
        "styleLibraryName": "theme-chalk"
      }
    ]
  ]
}

# 国际化

S2PC-UI 组件内部默认使用中文，若希望使用其他语言，则需要进行多语言设置。以英文为例，在 main.js 中：

// 完整引入 S2PC-UI 
import Vue from 'vue'
import S2PC-UI  from '@tencent/s2pc-ui'
import locale from '@tencent/s2pc-ui/lib/locale/lang/en'

Vue.use(S2PC-UI , { locale })

或

// 按需引入 S2PC-UI 
import Vue from 'vue'
import { Button, Select } from '@tencent/s2pc-ui'
import lang from '@tencent/s2pc-ui/lib/locale/lang/en'
import locale from '@tencent/s2pc-ui/lib/locale'

// 设置语言
locale.use(lang)

// 引入组件
Vue.component(Button.name, Button)
Vue.component(Select.name, Select)

# 目前 S2PC-UI 内置了以下语言：

  • 简体中文（zh-CN）
  • 英语（en）
