# ··· 前言

摄影（英語：photography）是指使用某种专门设备进行影像记录的过程。

# 核心

  • 摄影是是用光的艺术，是用光在绘画
  • 摄影是减法，创作是加法
  • 美感与内容呈现

# （一）摄影是是用光的艺术，是用光在绘画

摄影的英文叫作photograph，意即用光线作画，光线就像是画画中的颜料一样，是摄影作品的基本元素。从某种意义上来说，摄影是一种记录，即通过感光元件将现实世界中的光线进行记录。

  • 寻找好光线
  • 创造想要的光线
  • 后期手法

# （二）摄影是减法，创作是加法

# 何为减法？

绘画的过程，是在一张空白画布上从无到有，一笔一笔颜料添加叠加的结果，是一个从0到1的过程！而摄影呢，则是要在万千世界的千姿百态中，去伪存真，选取最有意义的一个时间空间加以记录，是一个从3到1的过程，所谓的3指的是复杂！

所以，摄影是一个做减法的过程，我们在摄影的构图上，会尽可能采取简化的思维模式，即将对我们的思想表达有影响或无关联的元素，通过摄影手法从画面中取除。

# 何为加法

都说摄影是一门技术，又是一门艺术。所谓的技术，其实更多是指摄影的记录本质。技术好，则记录的更准确。记录是对现实的一种存证，体现的是客观性。

而摄影又是一种艺术，在对现实进行记录的同时，还体现了摄影者对于世界的理解，这便是一种创造。

摄影艺术来说，不能仅仅停留在记录上，更应该通过自己独特的视角与深刻的思想，给照片加上生命力！好的照片，一定能够承载作者的思想。

# （三）美感与内容呈现

决定一张照片的好坏不是使用了什么技术，而是照片最后的美感以及内容的呈现。

# 如何学好摄影?

  • 多总结
  • 多看优秀的摄影作品
  • 多跳出框架去思考

# （一）多总结

“夫取法于上，仅得其中；取法于中，不免为下。” --唐·李世民《帝范》

# （二）多看优秀的摄影作品

“常与同好争高下，不共傻瓜论短长。”

每个领域都有自己的圈子，千万不要闭门造车，固步自封。

风光摄影分类：

  • 流行风光摄影：重在风光美的展现，以及情绪、气氛的渲染，后期处理尺度比较宽
  • 自然风光摄影：重在前期拍摄的探索和发现，后期处理尺度较窄。代表《国家地理》
  • 艺术风光摄影：重在探索一些创新的表达方式，以及记录，反思和展现人类发展，环境变迁，自然进化的相关主题。

# （三）多跳出框架去思考

跳出器材、工具、技术层面的羁绊，从更高的角度去思考

# 摄影四重境界

王国维在《人间词话》讲，古今之成大事业、大学问者，必经过三种之境界

  • 第一重境界：昨夜西风凋碧树。独上高楼，望尽天涯路；
  • 第二重境界：衣带渐宽终不悔，为伊消得人憔悴；
  • 第三重境界：众里寻他千百度，蓦然回首，那人却在灯火阑珊处。

摄影四重境界：

  • 器材
  • 技术
  • 形式
  • 内容

初学者千万不要掉进器材和技术的泥潭！

昂贵的镜头和相机、复杂的后期技术，很多时候并不能变成大片，只能变成一张更清晰更鲜艳的废片。

“技术决定下限，审美决定上限。” -- 李涛

决定一张照片是否美观，并不取决于我们到底使用的是曲线、色阶、柔光圈还是镜像滤镜或者其他的后期处理，而是取决于该突出的地方有没有提亮，该淡化的地方有没有压暗，有没有通过提亮和压暗让照片变得更美。

但一味的追求美观、大气的视觉效果，又容易陷入一个华而不实的怪圈。

照片的视觉效果终究还是要为照片的内容和情感服务的。

在照片中应用构图法则、色彩搭配、视觉引导，不仅为了照片好看，更是为了突出主题和主体，强化情感气氛并传达场景故事。

器材服务于技术，技术服务于形式，形式服务于内容

# 后期处理

摄影后期处理，技术是基础，思路是关键。

内容决定形式，形式决定技术

# 如何看一张好照片

# （一）震撼力

取决于题材的陌生感、画面的冲击力，以及故事的感染力。

冲击力：取决于照片的兴趣点

  • 广角风光中：惯用夸张前景，给观者带来冲击力
  • 长焦风光中：主要突出光影的明暗冷暖，以及物体的细节特点。

# （二）情绪与氛围

运用各种视觉语言，把创作者的情绪和拍摄环境的氛围传达给观者

  • 长曝光：带来慢门效果，表现出安静
  • 暖色光强化：温暖感
  • 降低照片饱和度：可以突出环境的苍凉和雄壮
  • 地曝光度：黑夜错觉，孤独感，寂寞感
